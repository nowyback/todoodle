## 🔧 Installation

1. Download the installer from the download page
2. Run `Todoodle Setup 0.11.0.exe`
3. Follow the installation wizard
4. Launch Todoodle from your desktop or Start menu

## 📱 Features

- **Custom Themes** - Create and apply custom themes
- **Background Images** - Set custom backgrounds
- **Dark/Light Mode** - Toggle between themes
- **Categories** - Organize tasks by category
- **Progress Tracking** - Track task completion
- **PWA Support** - Use on mobile devices
- **Local Storage** - All data stored locally
