Todoodle Server Setup

For advanced users who want to run Todoodle with a backend server:

  • Node.js 16+ installed
  • npm package manager
  • Terminal/command prompt access
  • Local network access

Quick Setup:

# Clone the server branch
git clone --branch server https://github.com/nowyback/todoodle.git
cd todoodle

# Install dependencies
npm install

# Start the server
npm run server

# Access the application
# http://localhost:3001
