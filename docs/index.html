Todoodle

A minimalist todo app for developers

Features

  • Desktop App - Full-featured Electron application with themes and backgrounds
  • Console Version - Complete command-line interface for terminal users
  • Custom Themes - Personalize your workspace with custom CSS themes
  • Background Images - Set custom backgrounds that override theme backgrounds
  • Dark/Light Mode - Toggle between dark and light themes
  • PWA Ready - Works as a Progressive Web App on mobile devices
  • Categories - Organize tasks by category
  • Progress Tracking - Track completion progress for tasks
  • Local Storage - All data stored locally, privacy-focused
  • Lightweight - Fast and efficient with minimal resource usage

Quick Start

Desktop Users: Download the Windows installer and run it.

Console Users: Use the command-line version with npm run console

For Developers: Set up the development server with these commands:

git clone https://github.com/nowyback/todoodle.git
cd todoodle
npm install
npm run console    # Start console version
npm start         # Start desktop app
npm run server    # Start API server

Custom Themes

Create your own themes by dropping .theme.css files in the src/themes/ folder:

  • Copy src/themes/TEMPLATE.theme.template
  • Rename to your-theme.theme.css
  • Customize colors and styles
  • Select in app settings (Ctrl+B)

Available themes: Dark Matter, Fallout Terminal, Translucent, and more.
