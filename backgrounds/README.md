# Backgrounds Folder

**LOCAL FILES ONLY** - No internet URLs supported!

## Supported Formats
- JPG
- PNG  
- WebP
- GIF

## How to Use
1. Place images in this folder ONLY
2. In Todoodle, press Ctrl+B
3. Enter the filename (e.g., `./backgrounds/my-image.jpg`)
4. Click "Set Background"

## Examples
- `./backgrounds/dark-theme.jpg` (for dark mode)
- `./backgrounds/light-theme.jpg` (for light mode)
- `./backgrounds/nature.jpg` (works for both)

## Rules
- ✅ Local files ONLY
- ❌ NO internet URLs
- ❌ NO external links
- ✅ Place files in this folder

## Tips
- Use high-resolution images for best results
- Images should be at least 1920x1080
- Darker images work better for dark mode
- Lighter images work better for light mode
- Keep images under 10MB for performance
